Gradient squircle avatar with user initials (or a photo) — used in the dashboard header and profile.

```jsx
<Avatar initials="RK" />
<Avatar src="/uploads/patient.jpg" size={56} shape="circle" />
```

Defaults to a 42px primary-gradient squircle. `shape="circle"` for round; `gradient="teal|gold"` for alternate fills; pass `src` for a photo.
