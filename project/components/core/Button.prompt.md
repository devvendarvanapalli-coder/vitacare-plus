Gradient-filled action button matching VitaCare Plus's AppButton — use for any tap target that performs an action.

```jsx
<Button variant="primary" leadingIcon="login" fullWidth onClick={signIn}>Sign In</Button>
<Button variant="outlined">Demo: Login as Patient</Button>
<Button variant="danger" leadingIcon="sos">Call 108</Button>
```

Variants: `primary` (blue gradient + glow), `secondary` (teal gradient), `danger` (red SOS gradient), `outlined` (1.5px primary border), `ghost` (text only). Sizes: `sm` 40px, `md` 52px, `lg` 54px. Supports `loading`, `disabled`, `leadingIcon`/`trailingIcon` (Material Symbols Rounded glyph names). Press shrinks to 0.97.
